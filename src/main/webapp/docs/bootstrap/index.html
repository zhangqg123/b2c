按钮

按钮样式

按钮样式 class属性 在元素标签上添加以下属性【a、input、button等等】 描述【按钮宽度是自适应文字长短的】
class="btn" 没有样式的按钮
class="btn btn-default" 比起没有样式，要好看些，IE7下没有.default这个类
class="btn btn-success" 标识成功或积极的动作
class="btn btn-primary" 提供额外的视觉感, 可在一系列的按钮中指出主要操作
class="btn btn-info"> 标识警告或需要用户注意
class="btn btn-warning" 标识危险或潜在的带来负面影响的动作
class="btn btn-danger" 表示这个动作危险或存在危险
class="btn btn-inverse" 备用的暗灰色按钮，不依赖于语义和用途【有兼容性问题，不建议使用】

按钮大小

按钮大小 class属性 在元素标签上添加以下属性【a、input、button等等】 描述【按钮宽度是自适应文字长短的】
class="btn success btn-large btn-lg" 为了兼容IE78 btn-large btn-lg 同时使用
class="btn btn-success btn-small btn-sm" 为了兼容IE78 btn-small btn-sm 同时使用
class="btn btn-success btn-small btn-sm" 为了兼容IE78 btn-mini btn-xs 同时使用

其它支持

其它支持 在按钮元素上的其它支持 描述【按钮宽度是自适应文字长短的】
class="btn btn-success disabled" disabled="disabled" nternet Explorer 9 及更低版本的浏览器将会把按钮中的文本绘制为灰色，并带有恶心的阴影，目前还没有解决办法。
class="btn btn-success active" 个人觉得没什么太大用处

提示工具

这是一段文字是的！这是一段文字这是一段一段文字，我也知道这是一段文字，但是这段文字问什么会出现在这里呢？为什么？

需要注意的是，是用提示工具必须得手动初始化。这是为了性能。。。怎么初始化？

初始化方法就是

$(function () {
    $('[data-toggle="tooltip"]').tooltip()
})

HTML代码为：

在需要提示的元素标签上加上 data-toggle="tooltip" 和 data-original-title="需要提示的内容"

如果要改变提示的显示位置，在元素标签上加上 data-placement="left、top、right、bottom"

弹出框

简单的弹出框

注意，为了浏览器性能，弹出框同样需要手动初始化！初始化方法如下：

$(function () {
    $('[data-toggle="popover"]').popover()
})

HTML代码为：

在需要弹出框的元素上加入 data-toggle="popover"为这个元素添加弹出框

弹出框标题为 data-original-title="这是弹出框的标题！"

弹出框内容为 data-content="这是一个弹出框的内容！是的，这的确是弹出框的内容！"

其它用法详见 其它用法

可消失的弹出框

可消失的弹出框

需要在点击其它地方时隐藏弹出框，则需要在元素节点上添加 data-trigger="focus"属性即可

其它用法：

data-trigger="click" 点击触发【默认】

data-trigger="hover" 鼠标移入移出触发

data-trigger="focus" 获得和失去焦点触发

data-trigger="manual" 手动触发触发【目前我还没弄明白怎么用。。】

多个触发事件可以同时使用，但是不建议这么做。同时使用用法： data-trigger="click focus"

只要在中间加上空格就好了，手动触发不能同时使用。

自定义弹出框位置

左侧： data-placement="left"

顶部： data-placement="top"

底部： data-placement="bottom"

右侧： data-placement="right"
