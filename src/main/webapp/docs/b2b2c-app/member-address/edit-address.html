说明

新增收货地址

API

/api/mobile/address/edit.do

参数说明备注

addr_id收货地址id此api需要会员登录

name收货人姓名此api需要会员登录

province_id省份id此api需要会员登录

city_id城市id此api需要会员登录

region_id地区id此api需要会员登录

addr详细地址此api需要会员登录

mobile手机此api需要会员登录

示例代码

var url = getFullUrl('/api/mobile/address/edit.do'); var options = { url : url, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("修改失败，请稍后重试。"); } } }; e$("#address_form").submitForm(options);
