说明

删除一个收货地址

API

/api/mobile/address/delete.do

参数说明备注

addr_id收货地址id此api需要会员登录

示例代码

var url = getFullUrl('/api/mobile/address/delete.do'); var options = { url : url, data : {addr_id : 3}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
调用结果：
{"result":1,"message":"删除收货地址成功！","data":null}
