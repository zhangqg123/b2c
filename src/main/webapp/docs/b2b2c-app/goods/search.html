说明

商品搜索

API

/api/mobile/goods/search.do

参数说明备注

catid商品分类id无

store_id商家店铺id无

goods_name商品名称无

sort排序字段无

order正序or倒序无

示例代码

var url = getFullUrl('/api/mobile/goods/search.do'); var options = { url : url, data : {catid:4, goods_name:"大码男装"}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
