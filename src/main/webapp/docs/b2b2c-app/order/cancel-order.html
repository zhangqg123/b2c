说明

取消订单

API

/api/mobile/order/cancel.do

参数说明备注

sn订单号此api需要登录

reason取消原因此api需要登录

示例代码

var url = getFullUrl('/api/mobile/order/cancel.do'); var options = { url : url, data : {sn : 147623284554-1, reason: "app测试取消订单"}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("取消订单失败，请稍后重试。"); } } }; e$.ajax(options);
调用结果：
{"result":1,"message":"取消订单成功","data":null}
