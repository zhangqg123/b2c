说明

通过session获取购物车所有商品、商品所对应的商家、商家的配送方式等信息

API

/api/mobile/order/store-cart-goods.do

参数说明备注

无无无

示例代码

var url = getFullUrl('/api/mobile/order/store-cart-goods.do'); var options = { url : url, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
