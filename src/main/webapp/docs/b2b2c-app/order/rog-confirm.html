说明

确认收货

API

/api/mobile/order/rog-confirm.do

参数说明备注

orderid订单id此api需要登录

示例代码

var url = getFullUrl('/api/mobile/order/rog-confirm.do'); var options = { url : url, data : {orderid : 4}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
调用结果：
{"result":1,"message":"确认收货成功","data":null}
