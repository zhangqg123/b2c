说明

会员登录

API

/api/mobile/member/login.do

参数说明备注

username用户名无

password密码无

code验证码无

validate_type验证类型为了让原生与H5版本的app兼容,1=需要验证码 0=不需要

示例代码

var url = getFullUrl('/api/mobile/member/login.do'); var options = { url : url, data : {username:"food",passsword:"111111",code:1111,validate_type:1}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("登录失败。"); } } }; e$.ajax(options);
调用结果：
{"result":1,"message":null,"data":{"face":"","username":"food","level":"黄金会员"}}
