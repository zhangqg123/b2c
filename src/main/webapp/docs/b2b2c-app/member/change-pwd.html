说明

修改会员登录密码

API

/api/mobile/member/change-password.do

参数说明备注

oldpassword旧密码此api需要会员登录

password密码无

re_passwd确认密码无

示例代码

var url = getFullUrl('/api/mobile/member/change-password.do'); var options = { url : url, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("修改密码失败。"); } } }; e$("#password_form").submitForm(options);
调用结果：
{"result":1,"message":"修改密码成功"}
