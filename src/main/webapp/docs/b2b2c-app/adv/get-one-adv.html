说明

根据某个广告id获取广告信息

API

/api/mobile/adv/get-one-adv.do

参数说明备注

advid广告id无

示例代码

var url = getFullUrl('/api/mobile/adv/get-one-adv.do'); var options = { url : url, data : { advid : 3 }, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
