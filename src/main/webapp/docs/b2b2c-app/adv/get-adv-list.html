说明

获取广告列表

API

/api/mobile/adv!advList.do

参数说明备注

acid广告位id无

示例代码

var url = getFullUrl('/api/mobile/order!list.do'); var options = { url : url, data : { acid : 10 }, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
