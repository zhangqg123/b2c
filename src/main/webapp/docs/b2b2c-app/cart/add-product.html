说明

将一个货品添加到购物车

API

/api/shop/cart/add-product.do

输入参数说明备注

goodsid商品id无

productid货品id无

havespec是否有规格因为与pc用的是一个接口，1表示有规格，0表示没有

num添加数量无

store_id店铺ID无

showCartData0是否在返回的json中显示购物车，0表示否，1表示是

activity_id活动ID优惠活动ID，如果没有默认为0

示例代码

var url = getFullUrl('/api/shop/cart!addProduct.do'); var options = { url : url, data : {productid:407, havespec:1, num : 1, goodsid:401, showCartData:0, activity_id:0, store_id:17}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
调用结果：
showCartData值为“0”：{"result":1,"message":"添加成功","data":null}
