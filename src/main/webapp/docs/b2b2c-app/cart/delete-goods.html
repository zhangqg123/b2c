说明

删除购物车中某一个商品

API

/api/mobile/cart/delete.do

参数说明备注

cartid购物车id无

示例代码

var url = getFullUrl('/api/mobile/cart/delete.do'); var options = { url : url, data : {cartid : cartId}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
调用结果：
{"result":1,"count":1}
