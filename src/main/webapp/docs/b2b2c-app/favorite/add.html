说明

收藏一个商品

API

/api/mobile/favorite/add.do

参数说明备注

id商品id需要用户登录

示例代码

var url = getFullUrl('/api/mobile/favorite/add.do'); var options = { url : url, data : { id : 407}, success : function(data) { if (data.result == 1) { alert(JSON.stringify(data)); } else { e$.alert("收藏失败，请稍后重试。"); } } }; e$.ajax(options);
调用结果：
{"result":1,"message":"收藏成功！","data":null}
