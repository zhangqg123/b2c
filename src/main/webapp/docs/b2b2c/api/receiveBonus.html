领取优惠券

API地址：

/api/b2b2c/bonus!receiveBonus.do?store_id=1&type_id=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
store_id int 必须 店铺id
type_id int 必须 优惠券id
useTimeStart String 必须 使用开始时间
useTimeEnd String 必须 使用结束时间
type_name String 必须 1 优惠券名称
type_money Double 必须 1 优惠券面额
min_goods_amount Double 必须 1 买家消费满
create_num int 必须 1 发行量
limit_num int 必须 1 每人限领

返回结果

名称 类型 是否必须 默认值 描述
result JSON 是 result=1表示领取成功，result=0表示领取失败
