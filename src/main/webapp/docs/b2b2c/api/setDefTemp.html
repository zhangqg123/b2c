设置默认模板

API地址：

/api/b2b2c/transport!setDefTemp.do?tempid=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
tempid int 是 1 模板id
member StoreMember 是 店铺会员

返回结果

名称 类型 是否必须 默认值 描述
result JSON 必须 如果等于“0”，设置默认失败，如果等于“1”，那么表示设置默认成功
