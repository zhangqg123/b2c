商品管理API

提供了商品的管理的功能

API列表

  • add
    发布商品
  • edit
    编辑商品
  • deleteGoods
    删除商品(将商品添加至回收站)
  • underGoods
    下架商品
  • cleanGoods
    清除商品
  • checkProInOrder
    检验是否有订单买过商品
  • revertGoods
    还原商品
  • saveGoodsStore
    修改商品库存
  • search
    店内搜索商品
  • snIsExist
    判断商品货号是否存在

数据结构列表

  • Goods
    搜索商品结果
  • storecart
    搜索商品结果
