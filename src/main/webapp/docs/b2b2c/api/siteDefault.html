设置默认发货地址

API地址：

/api/b2b2c/storeDlyCenter!siteDefault.do?dly_center_id=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
dly_center_id int 必须 1 店铺发货地址id
member StoreMember 必须 1 店铺会员

返回结果

名称 类型 是否必须 默认值 描述
result JSON 是 result=1表示设置删除成功，result=0表示设置失败
