店铺商品标签API

提供了增加、修改、删除店铺商品标签的功能和增加、修改、删除店铺商品标签引用的功能

API列表

  • add
    添加店铺商品标签
  • edit
    修改店铺商品标签
  • delete
    删除店铺商品标签
  • deleteRel
    删除商品标签引用
  • saveRel
    添加商品标签引用
  • saveSort
    修改商品标签引用排序

数据结构列表

  • StoreTag
