修改商品

API地址：

/api/b2b2c/goods!edit.do

输入参数

名称 类型 是否必须 示例值 默认值 描述
cat_id int 是 分类id,如果不输入，则搜索全部的分类下的商品
type_id int 是 商品类型id
name String 是 商品名称
sn int 是 商品编号
mktprice Double 是 市场价格
price Double 是 商品价格
price Double 是 商品价格
cost Double 是 成本价
weight Double 是 商品重量(g)
haveSpec Double 是 有没有规格
intro String 商品描述
goods_transfee_charge int 是 1卖家承担运费，2买家承担运费
store_cat_id int 是 本店分类
market_enable int 是 1立即发布，0暂不发布/td>

返回结果

名称 类型 是否必须 默认值 描述
result JSON 必须 如果等于“0”，修改错误，如果等于“1”，那么表示修改成功
