订单API

提供了对订单的创建、取消、改变店铺的配送方式以及红包、改变收货地址、订单确认、订单支付、订单发货、修改配送费用、修改订单金额、修改收货人信息的功能

API列表

  • create
    创建订单
  • cancel
    取消订单
  • changeArgsType
    改变店铺的配送方式以及红包
  • changeAddress
    改变收货地址
  • confirm
    订单确认
  • pay
    订单支付
  • ship
    订单发货
  • saveShipPrice
    修改配送费用
  • savePrice
    修改订单金额
  • saveConsigee
    修改收货人信息
