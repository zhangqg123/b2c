添加商品分类

API地址：

/api/b2b2c/goodsCat!addGoodsCat.do

输入参数

名称 类型 是否必须 示例值 默认值 描述
store_cat_name String 必须 分类名称
store_cat_pid int 必须 分类id
store_sort int 必须 排序
disable int 1 是否禁用，1为不禁用，0禁用

返回结果

名称 类型 是否必须 默认值 描述
result JASON 必须 如果等于“1”，保存成功，如果等于“0”，那么表示保存失败
