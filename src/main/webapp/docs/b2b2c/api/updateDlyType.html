修改物流工具

API地址：

/api/b2b2c/dlyType!update.do?dlyname=1000&tplType=1&tempid=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
storeMember StoreMember 必须 店铺会员
store_id Integer 必须 店铺id
dlyname String 必须 模板名称
tempid int 必须 模板id
tplType String[] 必须 模板类型

返回结果

名称 类型 是否必须 默认值 描述
result JSON 是 result=1表示修改成功，result=0表示修改失败
