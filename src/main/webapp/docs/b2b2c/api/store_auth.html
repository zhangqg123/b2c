提交店铺认证信息

API地址：

/api/b2b2c/storeApi!store_auth.do?store_id=1&fsid_img=1&fslicense_img=1&store_auth=1&name_auth=1

输入参数

name_auth
名称 类型 是否必须 示例值 默认值 描述
store_id int 必须 店铺id
fsid_img String 必须 身份证图片
fslicense_img String 必须 营业执照图片
store_auth Integer 必须 店铺认证
name_auth Integer 必须 店主认证

返回结果

名称 类型 是否必须 默认值 描述
result JSON 是 result=1表示成功，result=0表示失败
