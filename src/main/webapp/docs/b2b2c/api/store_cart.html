购物车API

提供了购物车的添加商品（货品）、添加数量的功能

API列表

  • addGoods
    将一个商品或者货品添加到购物车
  • getCartData
    获取购物车数据
  • invalidGoods
    结算页前判断是否有失效商品

数据结构列表

  • CartItem
    购物车条目
  • OrderPrice
    订单价格信息
