店铺管理API

提供了查看用户名是否重复、申请开店、重试申请开店、修改店铺设置、检测身份证、修改店铺Logo、提交店铺认证信息的功能

API列表

  • checkStoreName
    查看用户名是否重复
  • apply
    申请开店
  • reApply
    重新申请开店
  • edit
    修改店铺
  • checkIdNumber
    检测身份证
  • editStoreLogo
    修改店铺Logo
  • store_auth
    提交店铺认证信息

数据结构列表

  • StoreDetail
    店铺条目
