edit 修改收货地址API

修改收货地址

API地址：

/api/shop/member-address/edit.do

输入参数：

参数类型是否必须示例值默认值备注

addr_idint必填要修改的收货地址id

nameString必填王宏俊收货人姓名

provinceString必填黑龙江省所在省

province_idint必填所在省ID

cityString必填鸡西市所在城市

city_idint必填所在市ID

regionString必填鸡冠区所在地区

region_idint必填所在地区ID

addrString必填黑龙江省鸡西市鸡冠区西建委5组详细地址

zipString必填158100邮编

telString手机和电话必填一项0467-2325125固定电话

mobileString手机和电话必填一项18746789011手机

def_addrString选填yes是否是默认地址,如果传递yes则为默认地址，不传递此值或其它值为非默认地址

remarkString选填备注

返回结果

参数类型备注

resultint为1表示添加成功，0表示失败

messageString为提示信息

dataString返回数据

调用结果：

{"result":1,"message":"修改成功","data":null}
