getCartData 获取购物车数据API

获取购物车数据

API地址：

/api/shop/cart/get-cart-data.do

输入参数：

参数说明备注

无

返回结果:

参数类型备注

resultint如果等于“0”，表示失败，如果等于“1”，返回total值和count值

messageString返回提示信息

dataString返回数据Map

totalString如果成功，返回total等于购物车总价

countString如果成功，返回count值，即购物车内商品总数

返回示例：

{"result":1,"message":null,"data":{"total":0.0,"count":0}}
