getTotal 购物车的价格总计信息API

购物车的价格总计信息

API地址：

/api/shop/cart/get-total.do

输入参数:

参数类型是否必须示例值默认值备注

无

返回结果:

参数类型备注

resultint如果等于“0”，调用错误，如果等于“1”，那么表示调用成功

messageString提示信息

dataString返回数据Map

goodsPriceDouble商品价格

orderPriceDouble商品价格

shippingPriceDouble配送费用

needPayMoneyDouble需要支付的金额

discountPriceDouble折扣价

weightDouble商品重量

pointInteger获得积分

discountItemString优惠项目

actDiscountDouble法案折扣

gift_idString礼品id

bonus_idDouble奖金id

返回示例：

{"result":1,"message":null,"data":{"goodsPrice":0.0,"orderPrice":0.0,"shippingPrice":0.0,"needPayMoney":0.0,"discountPrice":0.0,"weight":0.0,"point":0,"discountItem":{},"actDiscount":0.0,"gift_id":0,"bonus_id":0}}
