addGoods 商品添加至购物车API

向后台提交商品的ID和数量

API地址:

/api/shop/cart/add-goods.do

输入参数：

参数类型是否必须示例值备注

goodsidint必须266提交到服务器一个商品的ID

numint必须1提交到服务器此商品的数量

showCartDataint必须0是否在返回的json中显示购物车

返回结果：

参数类型备注

resultint如果等于“0”，表示货品不足，如果等于“1”，那么显示添加成功

messageString提示返回信息

dataString返回数据

调用结果：

{"result":1,"message":"添加成功","data":null}
