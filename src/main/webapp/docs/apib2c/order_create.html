create 创建订单API

创建订单，购物车必须有数据，如果当前用户登录则会将此订单记录为此会员的订单，如果没登录则会提示登录

API地址：

/api/shop/order/create.do

输入参数：

参数类型是否必须示例值备注

address_idint必须1收货地址ID

payment_idint必须1支付方式ID

shipping_idint必须1配送方式ID

cityString必须海淀区城市

needPayint必须670需要收费

provinceString必须北京省

province_idint必须1省id

receiptContentString必须办公用品收据内容

receiptTypeint必须1收据类型

regionString必须海淀区地区

region_idint必须453地区id

shipAddrString必须北京市海淀区地址

shipMobileString必须13811111111联系电话

shipNameString必须XXX姓名

shipZipString必须1000846邮编

typeIdint必须1类型

shipDayString可选项配送时间

remarkString可选项备注
