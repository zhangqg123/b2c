说明

获得订单总价

方法

/api/mobile/order!getTotalPrice.do

参数说明备注

无

示例代码

var url = getFullUrl('/api/mobile/order!getTotalPrice.do'); var options = { url : url, success : function(data) { if (data.result == 1) { var viewData = data.data; var price = viewData.needPayMoney; $("#price_p").html("总计：" + e$.formatMoney(price)); } else { e$.alert("获取资源失败，请稍后重试。"); } } }; e$.ajax(options);
调用结果：
{ "result": 1, "data": { "discountItem": {}, "discountPrice": 0, "goodsPrice": 507.64, "needPayMoney": 515.64, "orderPrice": 515.64, "point": 518, "shippingPrice": 8, "weight": 1.4 } }
