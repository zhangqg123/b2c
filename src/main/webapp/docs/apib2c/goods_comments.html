add 商品的评论和提出咨询API

评论和咨询

API地址：

/api/shop/commentApi/add.do

参数类型是否必须示例值默认值备注

goods_idint必须商品id

commenttypeint必须1,2评论类型,可选值：1或2，1为评论，2为咨询

contentString必须评论内容

fileFile选填评论的图片

返回结果:

参数类型备注

resultint如果等于“0”，评论错误，如果等于“1”，那么表示评论成功

messageString返回提示信息

dataString返回数据

返回示例：

{"result":1,"message":"发表成功","data":null}
