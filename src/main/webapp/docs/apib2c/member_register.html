说明

会员注册

方法

/api/mobile/member!register.do

参数说明备注

username用户名

password登录密码

validcode验证码验证码同PC版验证码调用。

示例代码

var url = getFullUrl('/api/mobile/member!register.do'); var options = { url : url, data : {username : name,password : password,validcode : code}, success : function(data) { if (data.result == 1) { var param = e$.getParam(); if (param.script) { e$.execScript(param.script); } e$.closeWin({name:"login"}); } else { e$.alert("注册失败，" + data.message); } }; e$.ajax(options);
调用结果：
{"result":1,"message":"注册成功"}
