  • ·买家购买商品生成的所有订单，都会出现在该商品列表中，其中状态有订单状态、付款状态、发货状态。
  • ·订单状态分为:新订单,待确认、 已确认、 已取消、 已完成、 交易完成申请售后。
  • ·付款状态分为:已付款、部分付款、 未付款。
  • ·发货状态分为:未发货、 已发货、 已收货。
  • ·删除订单操作，只能删除作废订单，并且删除成功后，会放到回收站，您可以对其进行彻底清除或者还原其订单。
